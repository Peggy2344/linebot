### 使用說明
  機器人有泰英雙向翻譯及新聞查詢的功能 

  ##### 翻譯功能
  * 輸入要搜尋的單字即可，機器人會自動判斷語言並回覆內容 
  * 泰文翻譯輸入範例 : ข้อมูล
  * 英文翻譯輸入範例 : provide
  ##### 新聞查詢功能
  * 輸入關鍵字 : 查新聞 或 種類(頭條、商業、娛樂、健康、運動、科技)，機器人會回覆該種類的五則相關新聞
